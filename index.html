MarkItDown

Navegación

Instalación y Requisitos

Prepara tu entorno para utilizar MarkItDown. Requiere Python 3.10+.

Instalación Completa

Instala la herramienta con todas sus dependencias opcionales (recomendado).

pip install 'markitdown[all]'

Instalación Modular

Si prefieres un entorno más ligero, instala solo lo que necesitas (ej. PDF, Word, PowerPoint).

pip install 'markitdown[pdf,docx,pptx]'

Línea de Comandos (CLI)

La forma más rápida de convertir documentos directamente desde tu terminal.

Conversión Básica

Convierte un archivo y especifica el nombre del archivo de salida usando la bandera -o.

markitdown documento.pdf -o resultado.md

Uso con Tuberías (Pipes)

Puedes pasar el contenido de un archivo a través de la entrada estándar (stdin).

cat documento.pdf | markitdown > resultado.md

Uso Avanzado y Docker

Características adicionales como integraciones en la nube o ejecución aislada.

Azure Document Intelligence

Utiliza la API de Azure para extraer texto con mayor precisión y manejo de estructura.

from markitdown import MarkItDown

md = MarkItDown(docintel_endpoint="")
result = md.convert("documento_complejo.pdf")
print(result.text_content)

Uso mediante Docker

Ejecuta MarkItDown en un contenedor aislado, evitando conflictos de dependencias.

docker run --rm -i markitdown:latest < documento.pdf > resultado.md
